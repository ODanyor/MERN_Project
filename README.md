# The MERN Stack project

This is a little todolist project wich includes such kind of used tool as React, Redux, NodeJs(express), MongoDb.

## Getting Started

> git clone https://github.com/ODanyor/MERN_Project.git
> npm install
> npm run dev

## Built With

- React
- Redux
- NodeJS
  - Express
- MongoDb
