# The MERN Stack project

This is a little todolist project which includes such kind of tool as React, Redux, NodeJs(express), MongoDb.

## Getting Started

<pre>npm run dev</pre>

## Built With

- React
- Redux
- NodeJS
  - Express
- MongoDb
